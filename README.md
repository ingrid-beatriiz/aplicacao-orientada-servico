# 📚 Aplicação Orientada a Serviço

Repositório dedicado ao armazenamento das atividades, exercícios e projetos desenvolvidos durante a disciplina de **Aplicação Orientada a Serviço**.

## 🚀 Tecnologias e Ferramentas
As principais tecnologias abordadas neste repositório incluem:
* **Node.js & NPM**
* **Express**
* **Vercel**

---

## 📂 Estrutura do Repositório

### 1. Gerenciamento de Pacotes (NPM)
Atividade introdutória baseada no curso *Managing Packages with NPM* do freeCodeCamp.
* **Objetivo:** Criação e configuração de um arquivo `package.json` com metadados do projeto.
* **Arquivo principal:** `package.json`

### 2. Publicação no Vercel (API com Express)
Criação de um aplicativo Express mínimo com endpoints específicos e deploy na plataforma Vercel.
* **Objetivo:** Construir uma API REST simples com rotas GET e publicar o serviço na nuvem.
* **Endpoints:** * `/random`: Retorna um número aleatório entre 1 e 100.
  * `/dado`: Retorna um número aleatório entre 1 e 6.
  * `/citacoes`: Retorna um objeto JSON contendo uma citação aleatória de um cientista famoso e seu respectivo autor.
* **Pasta do projeto:** `atividade-express`
